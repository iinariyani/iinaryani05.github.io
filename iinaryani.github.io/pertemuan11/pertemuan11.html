Jawaban

  1. Apa gunanya CSS?

    IFRAME (Inline Frame) adalah elemen HTML yang digunakan untuk menampilkan halaman web atau konten lain di dalam halaman web yang sedang dilihat. Ini memungkinkan pengembang web untuk menyisipkan halaman web atau media eksternal seperti video, gambar, atau dokumen dalam halaman utama. IFRAME berguna untuk menampilkan konten dari sumber eksternal di dalam halaman web tanpa mengarahkan pengguna ke halaman eksternal tersebut.
